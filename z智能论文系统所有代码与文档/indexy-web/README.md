# indexy-web

前端工程：爬虫管理端界面，SPA应用，采用前后端分离架构

## 快速开始

- 环境

  安装node，然后执行
  
 `npm i -g restackx-cli`
 
 
- 安装依赖

 `npm i`

- 启动项目

 `npm run start`

- 构建项目

 `npm run build-prod`
 
- 生产模式运行

 `npm run build-prod` (若之前已经build过，则不用再执行build-prod)
 
 `npm run prod`

- 项目打包上线

 `npm run build-prod` 并且将打包后的文件夹移动到 ../pager-crawler/app/public (https://gitee.com/ai-pager/pager-crawler)
